## 자동화 도입. 
  어제부터 Claude AI를 적극적으로 사용하기 시작했다. 결제는 지난주 금요일에 했는데, 주말간 내가 기절하듯 잤던 것이 가장 큰 원인이다.   
 
  우선 먼저 시도해본건 주식분석 자동화다. 
  관련해선 Youtube에 영상들이 제법 많은데, 똑같이 해보려 해도 뭔 안되고 뭐는 되고 말이 많다.   
  pykrx와 yfinance가 가상공간에서 안된다는 오류때문에 고생을 좀 했는데, 결국 찾은 방법이 Github 의 Action을 이용하는 방법이었다.   

  스크립트를 작성해서 Github 액션을 통해 자동실행을 하면 결과값이 git 리포지터리에 저장되고,   
  윈도우에서는 스케줄링 실행을 통해 Pull 해오면, 
  그걸 클로드 cowork에서 작업해서 브리핑파일을 작성하고 카톡 알림을 준 다음, 
  몇분 후 일정 시각에 윈도우 스케줄링을 통해 Push 를 하는 방식으로 작성했다.  

  갖은 꼼수가 난무한다.그래서 그런가.. 아직까진 조금 더 손을 봐야 하나보다. 오늘아침에 실행되나 봤더니 윈도우가 멍때려서 실행을 안했더라... 전원옵션 조절하고, 내일아침에 실행하는지 봐야겠다.   
  
  이걸 자동화한게 오픈클로인가 싶다. 좀 더 고민해봐야겠다.
